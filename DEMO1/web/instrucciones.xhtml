    Llenar el Registro de Paciente.
    Llenar el Catálogo de Pais primero, para poder llenar el catálogo de Laboratorio.

    Llenar el Catálogo de Tipo de Vacuna, para poder llenar el Inventario.
    Llenar el Inventario de Vacuna, para poder llenar el Catalogo de Aplicación.
